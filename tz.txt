Техническое задание для "проекта перед работой".

Названия у проекта нет, далее - проект или сервис.

Проект должен предоставлять функционал для создания эвентов и поиска компании 
для совместного посещения или аудитории. 

В проекте должны быть функции:

1.1 регистрации и входа/выхода пользователя
1.2 создания и редактирования постов об эвентах
1.3 взаимодействия между между пользователем и сервисом
1.4 демонстрации профиля пользователей
1.5 api
1.6 администрирование сервиса

В проекте должны быть роли:

2.1 Администратора
2.2 Модератора
2.3 Пользователя


СЕРВИСЫ

(важные пункты помечены !, остальные - *)

1.1 регистрация

Регистрация как на vas3k.club, с интро. Вход либо по логину и паролю, либо по
уникальному коду, отправляемую на почту.

1.2 создание постов

Форма для создания постов должна включать в себя:

   ! титул                          # эти блоки должны отображаться  
   ! краткое описание (опционально) # в листинге событий на 
   * тэги для события               # главной странице сервиса
   ! стоимость                      # 

   ! развернутое описание события
   * количество участников (опционально)
   ! ссылка на чат события

1.3 взаимодействие пользователя с сервисом

   * сортировка и поиск событий по тэгам, стоимости, кол-ву
     участников на главной странице сервиса
   
   * возможность плюсовать и минусовать эвенты
   ! просмотр главной страницы сервиса с листингом всех эвентов, 
     переход на детальные страницы каждого эвента

1.4 профили пользователей

   * редактирование собственного профиля
   * просмотр профилей других пользователей (хоста эвента или его аппрувнутых 
                                             участников)

РОЛИ

2.1 Администратор
   
   Единственный в своем роде. Может раздавать, отнимать модерки. Имеет все фичи
   модераторов и обычных юзеров.

2.2 Модератор
 
   Подтверждает регистрации пользователей, подтверждает выход постов или 
   отправляет их на доработку. Блокирует неугодных. Имеет все фичи обычных 
   пользователей.

2.3 Пользователь

   Может создавать собственные эвенты (требует подстверждения от модератора), 
   просматривать чужие эвенты, откликаться на них, ставить плюсы и минусы 
   эвентам, редактировать свой профиль.
